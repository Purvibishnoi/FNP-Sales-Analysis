
# FNP Sales Analysis Dashboard

## Overview
This project analyzes 1,000 sales orders from **Ferns N Petals (FNP)** to identify key business insights across revenue, customer spending, product categories, and delivery performance. The goal is to support data-driven decision-making for improving sales, marketing, and logistics.
