#  FNP Sales Analysis (Excel Dashboard)

An interactive Excel-based data analysis project examining sales, customer behavior, and product performance for **Ferns N Petals (FNP)** across various occasions and cities.  
The goal is to uncover data-driven insights that help improve sales strategy, delivery performance, and customer satisfaction.

---

##  Project Overview
This project analyzes **1,000 sales orders** from FNP, focusing on:
- Revenue trends by occasion, product category, and month  
- Customer spending behavior and delivery performance  
- Regional sales insights across top-performing cities  
- Product-level performance and time-based ordering patterns  

The dashboard was created entirely in **Microsoft Excel** using **Power Query**, **Pivot Tables**, and **data visualization techniques**.

---

##  Key Insights
- **Total Revenue:** ₹35,20,984  
- **Average Order Value:** ₹3,520.98  
- **Average Delivery Time:** 5.53 days  
- **Top Occasions by Revenue:** Anniversaries and Raksha Bandhan  
- **Top Categories:** Colors and Soft Toys  
- **Top Cities:** Dhanbad and Imphal  

These findings highlight opportunities to improve delivery speed, strengthen Diwali campaigns, and target peak sales hours (12 PM & 6 PM).
